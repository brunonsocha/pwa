## Założenia aplikacji
Pomysł na aplikację jest prosty - aplikacja pozwalająca zrobić zdjęcie i nagrać do niego wiadomość głosową, po czym pozostawić oba na mapie. Inni użytkownicy mogą nacisnąć pinezkę, zobaczyć zdjęcie oraz odsłuchać wiadomość. Pomysł jest połączeniem heat-mapy ze Snapchata, ale bez losowania wyświetlanej treści ze wszystkich udostępnionych w danym regionie.
