## Założenia aplikacji
Pomysł na aplikację jest prosty. Użytkownik, spacerując, może pozostawiać "wiadomości" w postaci wiadomości głosowych oraz obrazów przypiętych do pinezek. Pinezki te, pojedyncze lub wiele, mogą zostać udostępnione komuś innemu, kto, w celu odsłuchania wiadomości lub zobaczenia zdjęć, będzie musiał zbliżyć się do nich na określoną odległość. 

Obecna wersja prezentacyjna obejmuje tylko obsługę odczytu lokalnie zapisanych plików, udostępnianie obejmuje same koordynaty, które w przyszłości pobierałyby z chmurowej bazy faktyczne pliki.
